# numerical-methods-interpolation
# Lagrange vs Newton Interpolation in Wolfram Language

This project compares two interpolation methods used in Numerical Methods:

- Lagrange Interpolation
- Newton Interpolation

Both methods are applied to the same dataset in Wolfram Language to show that they produce the same interpolation polynomial.
